Chapter 4

One Random Variable

本章把焦點放在單一隨機變數的完整描述方式：從累積分布函數、機率密度函數、期望值與常見連續分布開始，再進一步處理函數轉換、不等式、轉換方法、可靠度、亂數產生與 entropy。

Core Notes

本章基本內容

本章的核心任務，是把一個隨機量從「如何描述」推進到「如何計算、轉換、模擬與解釋」。閱讀時可把每一節都放回 CDF、PDF、期望值與工程意義之間的關係來理解。

描述分布 CDF、PDF、常見連續分布與參數意義。
計算特徵 期望值、變異數、動差與機率界限。
工程應用 可靠度、亂數模擬、entropy 與不確定性解釋。
4.1

The Cumulative Distribution Function

累積分布函數 CDF 描述隨機變數小於或等於某個數值的機率，是連續、離散與混合型隨機變數都能使用的統一表示法。學習重點包含 CDF 的單調性、極限行為，以及如何由 CDF 讀出區間機率。

4.2

The Probability Density Function

機率密度函數 PDF 用來描述連續隨機變數的機率分布形狀。這一節會強調密度不是單點機率，而是透過積分取得區間機率，並連結 CDF 與 PDF 之間的微分與積分關係。

4.3

The Expected Value of X

期望值提供隨機變數的平均趨勢，變異數與標準差則描述分散程度。課程會把 mean、moment、variance、standard deviation 與工程量測中的平均功率、雜訊強度等概念連在一起。

4.4

Important Continuous Random Variables

這一節整理常見連續分布，例如 uniform、exponential、Gaussian、Erlang 等。重點不只是記公式，而是理解每個分布適合描述哪一類工程現象，以及其參數如何影響分布形狀。

4.5

Functions of a Random Variable

當隨機變數經過函數轉換後，新的隨機變數也會產生新的分布。學習重點包括單調轉換、多對一轉換，以及用 CDF 或 PDF 方法求出轉換後的分布。

4.6

The Markov and Chebyshev Inequalities

Markov 與 Chebyshev 不等式提供在分布資訊不足時估計尾端機率的方法。這類界限常用來判斷偏離平均值的機率上限，是理解可靠度、風險與長期平均的重要工具。

4.7

Transform Methods

轉換方法把分布資訊轉到另一個表示空間，例如 moment-generating function 或 characteristic function。它能簡化期望值、動差與隨機變數和的分析。

4.8

Basic Reliability Calculations

可靠度計算把隨機變數用在元件壽命、失效率與系統存活機率上。這一節會把 CDF、survival function 與 exponential lifetime model 連結到工程系統可靠度。

4.9

Computer Methods for Generating Random Variables

電腦模擬需要能產生符合指定分布的樣本。本節介紹由 uniform random number 出發的產生方法，包含 inverse transform 的核心觀念與模擬驗證。

4.10

Entropy

Entropy 衡量隨機變數的不確定性，也連結資訊量與編碼效率。這一節會以直覺、公式與簡單分布例子說明 entropy 如何反映資料的不確定程度。

Summary

Chapter Summary

第 4 章完成從單一隨機變數的分布描述、期望與變異，到轉換、可靠度、模擬與 entropy 的基礎地圖。這些工具會在後續多變數隨機變數、隨機程序與訊號處理中反覆出現。
